# Pose Lab

Pose Lab is a mobile-first 3D posing and animation tool focused on touch workflows, reusable pose libraries, rapid iteration, and accessible animation authoring.

## Vision

Most animation tools assume:

- A desktop workstation
- Mouse and keyboard
- Long editing sessions
- Complex animation pipelines

Pose Lab explores a different idea:

> Animation should be possible anywhere.

On a couch.
On a phone.
In a waiting room.
Outside.

The goal is to make creating, saving, reusing, and sharing poses dramatically faster than traditional workflows.

## Current Direction

- Touch-first posing
- FK and IK manipulation
- Reusable pose libraries
- Cross-character pose reuse
- Animation composition from saved poses
- Retargeting workflows
- Mobile-friendly editing

## Long-Term Goals

- Personal pose libraries
- Shared pose libraries
- Animation snippets
- Retargetable motion assets
- Accessibility-focused workflows
- AI-assisted animation tooling

## Philosophy

A pose is knowledge.

If an animator creates a useful pose once, they should not need to recreate it from scratch.

Pose Lab treats poses, motion fragments, and animation decisions as reusable building blocks.

## Status

Experimental.

The project is under active exploration and may change significantly as workflows are tested and refined.

Contributions, feedback, and workflow ideas are welcome.
